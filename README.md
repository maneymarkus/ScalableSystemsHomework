## About The Project

An easy-to-use Todo-List where you can manage your todos and have a nice representation of your todos.


### Built With

* [Bootstrap](https://getbootstrap.com)
* [Django](https://www.djangoproject.com/)
* [Python](https://www.python.org/)



<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* Python
Download lastest version at https://www.python.org/downloads/ and follow the installation instructions

* npm
```sh
npm install npm@latest -g
```

* Django
```sh
pip install django
```

* Bootstrap
```sh
npm install bootstrap
```


### Installation
 
1. Clone the repo
```sh
git clone https://maneymarkus/ScalableSystemsHomework.git
```
2. Start the server
```sh
YourPathToTheProject\ScalableSystemsHomework\myproject python manage.py runserver
```



<!-- USAGE EXAMPLES -->
## Usage

Open http://localhost:8000/ in any browser and you will see your Todo-List and can start using it.
